TL;DR

Hosting on Mlab

OK, now we are done with the coding and it‚Äôs time to publish our updated application and the database it uses. To do so, we are going to use Mlab for the database and, again, Herokuapp for the application itself.

If you have followed the coding part of this tutorial, you will have to add a collection into the database and name it ‚Äòtweets‚Äô, in order to stay aligned with the code and get a working application in the end.
